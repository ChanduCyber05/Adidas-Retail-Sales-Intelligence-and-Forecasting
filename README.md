# Adidas Retail Sales Intelligence and Forecasting
Adidas Sales Intelligence Dashboard & Forecasting is an end-to-end data analytics project focused on uncovering key sales insights and predicting future performance. The project combines data cleaning, exploratory analysis, interactive dashboarding, and time-series forecasting to help stakeholders understand revenue trends, product performance, and regional sales patterns.  Built using modern analytics tools, this project demonstrates how data can be transformed into actionable intelligence for smarter, data-driven business decisions.
